La etiqueta html `<template>` es una manera de conservar html sin que se muestre al usuario para usarlo m√°s adelante mediante Javascript.

## Como usarla

Se debe colocar el contenido que ser√° usado dentro del elemento `template` y localizar el `content` con Javascript. En este caso lo usaremos para colocar items dentro de la lista.

```html
<ul id="list"></ul>

<template id="item">
  <li>
    <section>
      <h2 class="title"></h2>
      <p class="text"></p>
    </section>
  </li>
</template>
```

Debemos especificar la propiedad `content` para usar el contenido que esta dentro de la etiqueta que es el que nos interesa, no la etiqueta en si misma.

```javascript
const list = document.getElementById('list')
const item = document.getElementById('item').content
```

Luego el contenido del elemento `template` se tiene que clonar, se pueden usar estas 2 formas y obtendr√≠amos el mismo resultado.

Con cloneNode:

```javascript

```

Con importNode:

```javascript
const clone = document.importNode(item, true)
```

Y as√≠ podemos modificar los elementos localizandolos y posteriormente insertandolos al documento, esto con el fin de agregar la informaci√≥n que queramos, articulos, listas, datos de clientes, etc.

```javascript

title.textContent = 'Title 1'

// Etiqueta p
const text = clone.querySelector('.text')
text.textContent = 'Lorem ipsum dolor sit amet'

// Agregar a la lista
list.appendChild(clone)
```

## C√≥digo de ejemplo usando `fragment`

```html
<ul id="list"></ul>

<template id="item">
  <li class="list__item">
    <section class="card">
      <h2 class="card__title"></h2>
      <p class="card__text"></p>
    </section>
  </li>
</template>
```

```javascript
const list = document.getElementById('list')
const itemTemplate = document.getElementById('item').content
const fragment = document.createDocumentFragment()

const items = [
  { title: 'Title 1', text: 'Lorem ipsum dolor sit amet' },
  { title: 'Title 2', text: 'Lorem ipsum dolor sit amet' },
]

items.forEach((item) => {
  const clone = itemTemplate.cloneNode(true)

  const title = clone.querySelector('.card__title')
  title.textContent = item.title

  const text = clone.querySelector('.card__text')
  text.textContent = item.text

  fragment.appendChild(clone)

  list.appendChild(fragment)
})
```
